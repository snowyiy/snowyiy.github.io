Py-Apps est une application python disponible sur MacOS et composé de
(pour l'instant) 2 programmes : une calculatrice et un éditeur de texte.
Il y a également un script applescript pour les lancer automatiquement.

ATTENTION ! les programmes Py-Apps sont des programmes python il est donc nécessaire
d'installer python launcher au préalable et de le séléctionner comme application
par défaut pour les programmes python.
